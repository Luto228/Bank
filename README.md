# 🏦 BankProject

Добро пожаловать в **BankProject** — простую, но мощную систему управления банковскими счетами, написанную на Python с глубоким применением принципов ООП.

---

## 🚀 Основные возможности

- ✨ **Создание аккаунтов**: Генерация уникальных ID для каждого пользователя.
- 💰 **Управление балансом**: Безопасное пополнение и снятие средств.
- 🔒 **Безопасность**: Защита доступа к балансу и операциям с помощью пароля.
- 🔍 **Поиск**: Быстрый поиск аккаунтов в системе по ID.

---

## 🛠 Технологический стек

- **Язык**: Python
- **Библиотеки**: 
    - `random`: Используется для генерации уникальных 8-значных идентификаторов аккаунтов.

---

## 🏗 Столпы ООП в проекте

В этом проекте активно используются фундаментальные принципы объектно-ориентированного программирования:

### 1. 🧬 Инкапсуляция
Мы скрываем внутренние данные (состояние) объекта от прямого доступа извне. 
- Атрибуты `__password` и `__balance` в классе `Account` являются **приватными**.
- Доступ к ним осуществляется только через специальные методы (`deposit`, `withdraw`, `get_balance`), что гарантирует целостность данных.

### 2. 🏛 Классы и Объекты
- **Класс `Account`**: Представляет модель счета пользователя с его персональными данными.
- **Класс `Bank`**: Выступает в роли менеджера, который управляет коллекцией аккаунтов (инкапсулирует логику поиска и создания).

### 3. 🧩 Абстракция
Пользователь взаимодействует с высокоуровневыми методами банка (например, `createAccount`), не вникая в детали того, как именно генерируется ID или как обновляется список аккаунтов внутри системы.

## 💡 Пример использования

```python
from Main import Bank

# Создаем банк
my_bank = Bank()

# Создаем аккаунт
account = my_bank.createAccount('Alex', 1234)

# Пополняем баланс
account.deposit(1000)

# Снимаем деньги с проверкой пароля
account.withdraw(500, 1234)
```
